- 실무에서 자바스크립트를 타입스크립트로 변경할때

    기능적인 변경은 절대 하지 않을 것

    테스트 커버리지가 낮을 땐 함부로 타입스크립트를 적용하지 않을 것

    처음부터 타입을 엄격하게 적용하지 않을 것 (점진적으로 strict 레벨을 증가)

- //@ts-check 자바스크립트에 타입스크립트 처럼 문법 체크 해볼 수있음
